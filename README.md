## OLED模块
该模块代码源自江协科技开源的OLED代码，具体资料下载可登录江协科技官网：https://jiangxiekeji.com/
代码理论支持0.96英寸、1.3英寸的OLED屏幕，在此说明一下移植可能遇到的问题：
1. 江协科技采用软件SPI或软件I2C进行通信，移植时我们最好用硬件支持的SPI或I2C
2. 源代码适配于0.96英寸屏幕，若想用于1.3英寸屏幕，需要在`void OLED_SetCursor(uint8_t Page, uint8_t X)` 内修改一下光标偏移，否则会出现屏幕白条
## MusicPlayer
该代码源自学长传承，未作修改和维护，在数字钟系统内部套用部分代码逻辑，完成了利用Timer定时中断播放蜂鸣器的功能代码
音乐播放有三种实现方案：
1. 利用delay函数，逐个音节进行播放。该方案优点是实现简单，不需要调节PWM相关参数，但缺点是会阻塞主程序
2. 利用Timer的PWM输出和定时中断。启用两个Timer，一个输出PWM，一个定时中断用于调节PWM。该方案不会阻塞主程序
3. 除这两个方案之外，还可以利用DMA来控制播放，此处不展开
